Privacy Policy

This extension uses Supabase to store game-related data entered by users. We take your privacy seriously and handle all data responsibly:

Data Collected:

Game progress, stats, or related data that users choose to store.

No personally identifiable information (like name, email, or contacts) is collected unless explicitly provided by the user.

How Data is Used:

Data is used solely to provide game features within the extension, such as saving and displaying progress.

Data Storage:

All user data is stored securely on Supabase.

We do not sell, share, or distribute any user data.

User Control:

Users can delete their data at any time through the extension.

Permissions:

The extension accesses certain public APIs and websites to display game information.

No additional personal data is collected from these sites.
